Installation


Installing BrowserCraft is Super-Easy.

All you need to do is download the ZIP file, by clicking the ZIP icon on the top left corner. Or by clicking download in the menu.

After downloading the files, extract them to any folder then run mcHTML.html .

Currently only Firefox and Google Chrome are supported. More browser support to come.

The ZIP files are being constantly updated so make sure you always download from time to time for the latest features.
